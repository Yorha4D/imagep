# imagep

个人图片托管仓库，存放博客、文档、视频脚本中可公开引用的截图素材。

**基础直链格式：**

```
https://raw.githubusercontent.com/Yorha4D/imagep/main/<文件名>
```

## 使用方法

**Markdown**

```md
![Blender 5.2 文字 UI](https://raw.githubusercontent.com/Yorha4D/imagep/main/blender-5.2-text-ui.png)
```

**HTML**

```html
<img src="https://raw.githubusercontent.com/Yorha4D/imagep/main/blender-5.2-text-ui.png" alt="Blender 5.2 文字 UI">
```
